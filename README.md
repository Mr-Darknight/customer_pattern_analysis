# customer_pattern_analysis
Customer Behavior Analysis project using **Python (Pandas), SQL, and Power BI** to clean, analyze, and visualize customer purchasing data. Built an interactive dashboard to explore insights such as revenue by category, sales trends, subscription distribution, and customer behavior patterns.
# Customer Behavior Analysis

## Overview

This project analyzes customer purchasing behavior using Python, SQL, and Power BI. It demonstrates a complete data analytics workflow including data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation.

## Dataset

The dataset contains customer information such as gender, category, purchase amount, review ratings, shipping type, and subscription status.

## Tools

* Python (Pandas, NumPy)
* SQL (PostgreSQL / MySQL / SQL Server)
* Power BI
* Gamma (for presentation)

## Steps

1. Load and clean the dataset using Python (Pandas).
2. Perform exploratory data analysis (EDA).
3. Store and query the data using SQL.
4. Build an interactive Power BI dashboard.
5. Create a report and presentation.

## Dashboard

The Power BI dashboard provides insights such as:

* Number of customers
* Average purchase amount
* Average review rating
* Revenue by category
* Sales by category
* Customer subscription analysis
* Seasonal revenue trends

## Results

The analysis helps understand customer behavior, revenue distribution, and purchasing patterns, enabling better business insights and decision making.

## How to Run

1. Load the dataset in Python and perform data cleaning.
2. Run SQL queries for analysis.
3. Open the Power BI file to explore the dashboard.
4. Review the report and presentation for insights.
